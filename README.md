# TRPGシナリオ コピー支援

TRPGシナリオ本文から、必要な行をクリックまたはドラッグでコピーする静的Webアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「入力・設定」で本文を直接入力するか、UTF-8の `.txt` ファイルを読み込みます。
3. 必要に応じて「置換設定（ココフォリア等の変数）」から置換ルールを追加します。
4. 「コピー」タブで行をクリックすると単行コピー、行をドラッグすると連続行コピーできます。

## 公開

依存関係やビルド手順はありません。
GitHub Pagesでは、リポジトリ直下を公開対象にするとそのまま動作します。
